HOW LIVE IDS WORKS
Simulate mode:
Generates realistic synthetic traffic patterns (70% normal, 12% DoS, 8% Probe, 5% R2L, 5% U2R). All 6 real models classify each flow in real-time.

Capture mode (Linux only):
Raw socket (AF_PACKET) on selected interface. Extracts TCP/UDP flow features from actual packets. Requires root privileges.

Alerts: CRITICAL/HIGH triggers instant Slack notification (configure SLACK_WEBHOOK_URL in .env).
